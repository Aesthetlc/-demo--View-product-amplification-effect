# View-product-amplification-effect
案例效果：原生JS-实现京东查看商品点击放大案例

## 需求分析

* 鼠标移入，出现黄色的遮罩，和用来放大的盒子
* 鼠标移出，遮罩和放大的盒子消失
* 鼠标在小图片上面进行移动的时候
  * 黄色遮罩层会随着鼠标一起移动
  * 用来放大显示的图片，也跟着一起移动
